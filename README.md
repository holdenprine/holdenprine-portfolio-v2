## Portfolio Site

This is my personal portfolio website, built using **Next.js** with **TypeScript** for robust and scalable development.  
The site showcases my projects, experience, and skills while delivering a seamless user experience through modern frontend technologies.

---

## 🚀 Features

- Developed with **Next.js** and **TypeScript** for optimized performance and type safety  
- Styled using **Tailwind CSS** for responsive, utility-first design  
- Smooth, dynamic animations powered by the **Motion Library**  
- Fully responsive and mobile-friendly layout  
- SEO-optimized for better search engine visibility  

---

## 🔧 Technologies Used

- **Framework:** Next.js  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** Motion Library (Framer Motion)
